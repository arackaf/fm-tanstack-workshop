Welcome to TanStack Jacked!

# Getting Started

Make sure you have Docker Desktop running (install it if needed).

Run

```bash
npm run pg
```

or

```bash
npm run pg-raw
```

if you want a fresh database whose contents will be lost when you close the terminal.

Then in a fresh terminal run

```bash
npm install
npm run dev
```

# Building For Production

To build this application for production:

```bash
npm run build
```
